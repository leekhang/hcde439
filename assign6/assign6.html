For this assignment, I implemented a potential divider to read in an input of a potentiometer and display it on a web browser via p5.js framework.

Schematic

In this circuit, A potentiometer is used in a potential divider to vary the analog input into the arduino. Since the pot was at a maximum of 10KΩ, I decided to use a smaller resistor (which was 1KΩ) so that I can read a greater range of values (which is between 0 to 930).

Arduino .ino File

Sketch.js File

The sketch file from the hcde439-example repo is used. The graphData function is used to display the values of the potentiometer overtime. The portName variable is changed to suit the port used by the Arduino.

Console

The value is printed onto the browser console using the "console.log()" function in the sketch.js file.
